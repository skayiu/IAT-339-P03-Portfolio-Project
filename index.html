Interactive Elements

In-text link

CSS selector:

a, .link

Sample code:

<a class = "link" href = "#"> Link </a>

Rendered element:

Link

Navigation Link

CSS selector:

nav a, .link

Sample code:

<nav>
<a class = "button" href = "#"> default link </a>
</nav>

Rendered element:

default link

Navigation Button

CSS selector:

a, .button-link

Sample code:

<a class = "button-link" href = "#"> default button </a>

Rendered element:

default button

Footer Navigation Link

CSS selector:

a, .footer-nav

Sample code:

<nav>
<a class = "footer-nav" href = "#"> style guide </a>
</nav>

Rendered element:

style guide

Navigation Image as a Link

CSS selector:

a, .nav-img

Sample code:

<a href = "home.html"> <img src = "img/logo.png" alt = "Logo" class = "nav-img"> </a>

Rendered element:

Project Image as a Link

CSS selector:

a, .project-img

Sample code:

<a href = "project-1.html"> <img src = "img/l233-sketch.png" alt = "Study Carrel Isometric Front" class = "project-img"> </a>

Combined Elements

Navigation Bar

CSS selector:

header, a, nav a, .nav-img, .button-nav

Sample code:

<header>
<a href = "home.html"> <img src = "img/logo.png" alt = "Logo" class = "nav-img"> </a>
<nav>
<a class = "button-nav" href = "about.html"> about </a>
<a class = "button-nav" href = "projects.html"> projects </a>
</nav>
</header>

Rendered element:

about projects

Page Banner

CSS selector:

.box-banner, h2, p

Sample Code:

<div class = "box-banner">
<h2> Page Title </h2>
<p> Put page subtext here. </p>
</div>

Rendered element:

Page Title

Put page subtext here.

Footer

CSS selector:

nav, ul, li, a, .footer-nav

Sample code:

<nav>
<ul class = "footer">
<li> <a class = "footer-nav" href = "home.html"> home </a> </li>
<li> <a class = "footer-nav" href = "about.html #contact"> contact </a> </li>
<li> <a class = "footer-nav" href = "https://github.com/skayiu/P03-Portfolio-Project"> p03 github </a> </li>
<li> <a class = "footer-nav" href = "index.html"> style guide </a> </li>
<li> <a class = "footer-nav" href = "index.html #citations"> citations </a> </li>
</ul>
</nav>

Rendered element:

  • home
  • contact
  • p03 github
  • style guide
  • citations

Visual Aspect

Colour Palette:

Main colour - #30A3FA

Used for h2, h3, h4, h5, & p text, button hover & borders

Sub colour - #0064B0

Used for h1 text on home page and project image as a link hover

Accent colour - #FFBBD0

Used for hover effects and highlighting content

Accent colour - #FFFFFF

Used for background colour

Fonts:

Title font: Waiting for the Sunrise

Used for h1 and h2 on bio

Main font: Poppins

Used for portfolio content
