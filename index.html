Interactive Elements

In-text link

CSS selector:

.link

Sample code:

<a class="link" href="#"> Link </a>

Rendered element:

Link

Navigation Link

CSS selector:

.button-nav

Sample code:

<a class="button" href="#"> Default Button </a>

Rendered element:

Default Button

Submit/Reset Buttons

CSS selector:

input[type=submit], input[type=reset]

Sample code:

<input type = "submit" value = "Submit" /> <input type = "reset" value = "Reset" />

Rendered element:

Text Input Field with Label

CSS selector:

.box_form, .box_form_item, label, input[type=text]

Sample code:

<form class="footer_box_form"> <div class = "box_form_item"> <label for = "Input"> Input: </label> <input type = "text" id = "Input" input = "Input"> </div> </form>

Rendered element:

Radio Button with Label

CSS selector:

input, label

Sample code:

<div class = "radio-item"> <input type = "radio" id = "input1" name = "option" value = "input1"> <label for = "input1"> Option 1 </label> <br> </div> <div class = "radio-item"> <input type = "radio" id = "input2" name = "option" value = "input2"> <label for = "input2"> Option 2 </label> <br> </div> <div class = "radio-item"> <input type = "radio" id = "input3" name = "option" value = "input3"> <label for = "input3"> Option 3 </label> <br> </div>
