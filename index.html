Interactive Elements

In-text link

CSS selector:

.link

Sample code:

<a class = "link" href = "#"> Link </a>

Rendered element:

Link

Navigation Link

CSS selector:

.button-nav

Sample code:

<a class = "button" href = "#"> Default Button </a>

Rendered element:

Default Button

Footer Navigation Link

CSS selector:

a, .footer-nav

Sample code:

<a class = "footer-nav" href = "index.html"> Style Guide Home </a>

Rendered element:

Style Guide Home

Navigation Image as a Link

CSS selector:

a, .nav-img

Sample code:

<a href = "home.html"> <img src = "img/logo.png" alt = "Logo" class = "nav-img"> </a>

Rendered element:

Project Image as a Link

CSS selector:

a, .project-img

Sample code:

<a href = "project-1.html"> <img src = "img/l233-sketch.png" alt = "Study Carrel Isometric Front" class = "project-img"> </a>

Headings

h1, h2, h3, h4

Numbered Lists

CSS selector:

ol, li

Sample code:

<ol> <li> Item 1 </li> <li> Item 2 </li> <li> Item 3 </li> </ol>

Rendered element:

  1. Item 1
  2. Item 2
  3. Item 3
